JavaScript Scope

Week 8 | 9 February 2017

What is scope?

Scope refers to the visibility of variables in a program. In other words, the extent to which parts of your program can see or use a variable. In Javascript there are three forms of scope: global, local and block.

If you've never heard of scope before, you can think of the "visibility" thing like fame.

Say you have two Taylors: one is Taylor Swift, and the other is Taylor Small. Taylor Small is known locally, but not beyond the borders of her town. Taylor Swift meanwhile is famous globally. People all around the world have heard of her.

How does it work?

A global variable is one that has been declared outside a function or block. Variables that are declared globally are visible in the local scope, but variables declared inside a function (local variables) aren't visible in the global scope.

E.g. 
var taylorSwift = "famous"

function fameCheck(name) {
  var taylorSmall = "not so famous"
  console.log(name)
}

fameCheck(taylorSwift) // famous
fameCheck(taylorSmall) // taylorSmall is not defined

In the example above, taylorSwift is a global variable, while taylorSmall is a local variable. When you try to call the function with the variable taylorSmall you get an error message, because taylorSmall isn't visible in the global scope.

Functions create local scope. Each function's scope is separate, so whatever you do in one function won't affect what you do in another.

So that leaves block scope. A "block" is a section of code, like the piece of code between the opening and closing braces of an if statement. You can use "let" or "const" to create block-scoped variables. For these variables, the braces {...} define a new scope. When you set one of these variables to a particular value, the value applies only to the scope in which it is set. The example below, taken from this resource, shows how let works and why it is useful:

let x = 10;
var y = 10;
{
    let x = 5;
    var y = 5;
    {
        let x = 2;
        var y = 2;
        document.write("x: " + x);
        document.write("y: " + y);
        // Output:
        // x: 2
        // y: 2
    }
    document.write("x: " + x);
    document.write("y: " + y);
    // Output:
    // x: 5
    // y: 2
}
document.write("x: " + x);
document.write("y: " + y);
// Output:
// x: 10
// y: 2
