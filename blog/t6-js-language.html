Five JS operators and when to use them

1. Parentheses ()

  • Grouping - allows expressions with lower precedence to be evaluated before an expression with higher priority. E.g. total = (5 + 5) * 5 equates to 50, as opposed to total = 5 + 5 * 5 which equates to 30.
  • Function call - used to call a function. If a number or value is included inside the parentheses, that value will be passed to the function. E.g. 
    function equation (a, b) {
  total = a + b
  console.log(total)
}

equation(4, 6) // 10
  • Placeholders for data

2. Brackets []

  • Property accessor - used to access an object's properties.
    e.g.1 when the property name is contained in a variable: 
    var crunchyFruit = "fruit1"

var fruit = {
  fruit1: "apple",
  fruit2: "banana",
  fruit3: "orange"
  }

var chosenFruit = fruit[crunchyFruit]
console.log(chosenFruit) // apple
    e.g.2 when the property name contains characters not permitted in identifiers (e.g. starts with a digit, or contains a space or dash): 
    var object = {
  "first item": "fork",
  "second item": "knife"
}
console.log (Object["first item"]) // fork
  • Arrays - used to notate an array e.g. 
    var array = [3,4,5];
array.length; //3

3. Braces {}

  • Objects - used to notate an object e.g. 
    var object = {
  item1: coffee,
  item2: cake
}
  • Block statements - commonly used with control flow statements such as for and if e.g. 
    if (hour < 18) {
    console.log("Have a nice day");
} else {
    console.log("Have a nice evening");
}

4. Single quotes ''

  • Where a string contains double quotes - e.g. alert('Say "Hello"');

5. Double quotes ""

  • Where a string contains single quotes - e.g. alert("Say 'Hello'");
