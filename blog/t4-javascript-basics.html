Arrays and objects

Arrays and objects are used to store multiple values in a single variable (var). For instance, if you wanted to store a range of muffin flavours in the variable "muffins", you could use an array as follows: 

var muffins = ["blueberry", "banana choc chip", "apple cinnamon"]
Numbers are used to access the values (or "elements") of an array. In this case, muffins[0] returns "blueberry" (arrays indexes start with 0).

An array is a special type of object. One major difference between an array and an object is that you use names to access the values in an object, rather than numbers. For instance, in the example below, cake.filling returns "jam": 

var cake = {type:"sponge", filling:"jam", number: 3};

Arrays are best when you want your element names to be numbers. Objects should be used when you want the element names to be text.

Functions

A function is a "subprogram", made up of a sequence of statements, which can be used to perform a particular task. As Code Academy puts it: "a function takes in inputs, does something with them, and produces an output."

Functions can be declared using: 

function nameOfFunction(parameter) {
  console.log("Print the word " + parameter);
};

To use a function, you "call" the function by typing the function's name and putting a parameter value inside the parentheses after it e.g. for the above example, function("Hello") would output "Print the word Hello".

Functions are useful because they allow you to use the same piece of code over and over without having to write the whole thing out again. You can use the same function with different values (or "arguments") and it will produce different results.
